# STEP400 demo sketch for Processing
This is a Processing sample sketch for the stepper driver STEP400 from [Ponoor Experiemnts.](https://ponoor.com/en/)

こちらは[Ponoor Experiemnts](https://ponoor.com/)さんのステッピングモータードライバー、STEP400用のProcessing用サンプルスケッチです。

## Set up & Wiring
Please go through [this tutorial](https://ponoor.com/en/docs/step400/) on the Ponoor's page.

セッティング、配線はこちらの[チュートリアルのページ](https://ponoor.com/docs/step400/)を参照してください。

## Library dependency
This sketch uses [Andreas Schlegel's oscP5 library](http://www.sojamo.de/libraries/oscP5/). It can be loaded to the sketch via Sketch>Import LIbrary...>Add Library...
Then from Libraries tab search oscP5.

こちらのスケッチでは[Andreas Schlegelの oscP5 library](http://www.sojamo.de/libraries/oscP5/)を使用しています。
スケッチメニュー＞ライブラリのインポートからoscP5を検索して追加してください。
